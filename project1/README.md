# Causal Impact Analysis with sklearn

This project analyzes the causal effect of a new marketing intervention (daily Twitter promotions) on a blog's daily visitor count. Instead of using specialized libraries, this analysis uses a robust and stable approach with scikit-learn to measure the impact.

Project Goal: To statistically measure the additional traffic (lift) generated by the Twitter promotions, separating its effect from normal daily trends and seasonality.

### methodology

The core idea is to model the "counterfactual"â€”what would have happened if the promotions had never started?

To do this, a LinearRegression model is trained only on the "Pre-Intervention" data (Jan 1 - Aug 31). The model learns the blog's "normal" traffic pattern based on its growth trend and seasonality.

It then predicts the traffic for the entire period, including September. This prediction serves as a baseline of what would have happened. The difference between this baseline and the actual traffic is the causal impact of the promotion.
